Batatinha Bot

Batatinha Bot Commands

Limpar o chat : .clear ou .limpar

Pingar everyone : .fakehere

Criar um canal : .canal + o nome do canal

COMANDOS para proteger o seu servidor

Fechar todos os canais : .lockdown

Abrir todos os canais : .unlockdown

Fazer a manutenção do servidor : .manutenção "bloquear todos os canais"

Retirar da manutenção : .manutençãooff "Isso vai desbloquar todos os canais"

Interagir com um beijo : .beijar ou .kiss

Interagir com um abraço : .abraçar ou .hug

Interagir com um soco : .soco ou .bater ou .slap

Escolher alguma coisa : .escolher algumacoisa1 algumacoisa2 (tudo junto)

Em breve irei adicionar mais comandos para sugestões entre no servidor.
